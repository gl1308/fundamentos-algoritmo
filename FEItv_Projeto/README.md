
# Projeto FEItv

Projeto desenvolvido em Python para a disciplina CCP110.

## Funcionalidades
- Cadastro de usuários
- Login
- Busca de vídeos
- Listagem de vídeos
- Curtir vídeos
- Favoritos

## Tecnologias
- Python
- Arquivos TXT

## Como executar
1. Instale o Python 3
2. Execute o arquivo:
   python main.py

## Arquivos utilizados
- usuarios.txt
- videos.txt
- favoritos.txt
